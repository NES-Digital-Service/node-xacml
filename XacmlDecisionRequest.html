(static) fromCompactForm(compactFormRequest)

Returns a new XACML request from a "compact form" JSON object.
Parameters:
Name Type Description
compactFormRequest Object The compact form is a JSON object where each object key is an XACML attribute category. Each object value is an Array, the elements of which are attributes in same the form as #addAttribute.
Source:
  • XacmlDecisionRequest.js, line 74
Example
let compactRequest = {
  category1: [ { id: 'attribute1-id', value: 'attribute1-value' } ],
  category2: [ { id: 'attribute2-id', value: 'attribute2-value' } ]
}
let request = XcamlDecisionRequest.fromCompactForm( compactRequest )

addAttribute(categoryId, attribute, includeInResultopt)

Adds the given attribute to this decision request.
Parameters:
Name Type Attributes Description
categoryId String the attribute category this attribute should be added to. To ensure compatability with PDP it is recommended to only use standard XACML categories defined in XacmlAttributeCategory.
attribute Object
Properties
Name Type Attributes Description
id String the id of attribute. Standard XACML attribute ids are defined in XacmlAttributeId.
type String <optional>
 the type of the attribute. The default is `http://www.w3.org/2001/XMLSchema#string`.
value String the value of the attribute.
includeInResult Boolean <optional>
 should this attribute be returned in the result. The default is `false`.
